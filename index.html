Full-Stack Web App · React · Supabase

CookieCommand

A complete, real-time cookie season management portal built for Girl Scout Troop 04326 (GSGLA). Handles inventory tracking across 38 scouts and 10 cookie varieties, peer-to-peer box trades with validation, real-time troop chat (global + DMs), booth scheduling, an interactive troop calendar, auto-notifications, and a full admin panel with financial reporting — all in one mobile-friendly React SPA. Runs in dual mode: cloud sync via Supabase Realtime (WebSocket) or pure localStorage fallback. Total cost: $0.

// 03 — About

The Narrative

"Ship it. Then make it better."

I'm Chad Keith — a full-stack developer and AI engineer who builds things people actually use. My work spans the full stack: React SPAs with Supabase real-time backends, FastAPI services with LangChain agents, Discord bots that literally speak to users and produce psychological assessments, ComfyUI tooling used by generative-AI artists, and websites with domain names that state the obvious.

CookieCommand started as a favour for a Girl Scout troop and turned into a full production app — 38 scouts, real-time inventory sync, troop chat, booth scheduling, and a financial dashboard that tracks every dollar across the cookie season. Zero ongoing cost.

StaffLens solves a real problem: community managers need to vet applicants but hate the process. The bot joins a Discord voice channel, has a real conversation, and delivers a structured hire/no-hire report — all without a human ever sitting in.

The Agentic RAG Demo Toolkit is proof-of-concept made deployable: a full LangChain + FastAPI + Next.js stack that demonstrates grounded chat, live document ingestion, and deterministic tooling — documented well enough to run in an interview in minutes.

ComfyUI-ChomfyNodes grew out of frustration with the limits of off-the-shelf nodes. 24-slot wildcard prompting, forensic image provenance, universal noise injection across every major model architecture.

I believe in sharp edges: clear APIs, obvious interfaces, code that does exactly what the name says. Things should be precise, honest, and direct.

If you need someone who can go from idea to deployed app — and explain every decision along the way — let's talk.
