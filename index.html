I'm Chad Keith — a full-stack developer and AI engineer who builds things people actually use. React SPAs, FastAPI services, LangChain agents, Discord bots that literally speak to users, ComfyUI tooling, and websites with domain names that state the obvious.

ConversaTrait is my flagship product: an augmented intelligence platform that performs Big 5 personality analysis through natural conversation — no questionnaires, no forced prompts. It reads how people communicate and surfaces psychological traits underneath. Used by hiring teams, coaches, and community managers.

CookieCommand started as a favour for a Girl Scout troop and became a full production app — 38 scouts, real-time Supabase sync, troop chat, booth scheduling, peer-to-peer box trades, and a financial dashboard. Deployed on GitHub Pages. Total cost: zero.

StaffLens solves a real problem: community managers hate sitting through applicant voice calls. The bot joins the Discord channel, has a real conversation, and delivers a structured hire/no-hire report. Deepgram STT, Edge-TTS, Claude via OpenRouter.

I believe in sharp edges: clear APIs, honest interfaces, code that does exactly what the name says.
